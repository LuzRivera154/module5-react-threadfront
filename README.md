# Instructions d’installation

## 1. Cloner le dépôt
```bash
git clone https://github.com/LuzRivera154/module5-react-threadfront.git

2. Installer les dépendances du backend

Accédez au dossier du backend :

cd back

Installez les dépendances principales :

npm install

Installez les dépendances supplémentaires :

npm install sequelize mysql2 express jsonwebtoken cors
npm install bcrypt

Revenez au dossier principal si nécessaire :

cd ..

3. Installer les dépendances du frontend

Accédez au dossier du frontend :

cd thread-front

Installez les dépendances :

npm install
